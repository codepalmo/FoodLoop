# FoodLoop: Sustainable Surplus Management System 🍏

FoodLoop is a Flutter-based mobile application engineered to bridge the logistical gap between urban food surplus and local community needs. By digitizing the "sharing economy," the app transforms potential landfill waste into a vital community asset through real-time coordination.

## 🚀 Key Features
- **Hyper-Local Discovery**: Real-time Google Maps integration for locating food surplus within a 5km radius.
- **NGO Enterprise Suite**: A dedicated interface for registered organizations to claim and manage high-volume donations.
- **Smart Expiry Tracking**: Automated system alerts to ensure food is claimed and consumed before the safety window closes.
- **Gamified Engagement**: A rewards engine using a `POINTS_TYPE` schema to incentivize community giving.

## 🛠 Tech Stack
- **Frontend**: Flutter Framework (Dart)
- **Backend**: Firebase (Cloud Firestore & Authentication)
- **APIs**: Google Maps Platform (Geocoding & Directions)
